# React Cart Application

## Description
A ReactJS shopping cart application built using Fake Store API.  
Users can view products, add/remove items from cart, update quantity, and see total price with discount.

## Tech Stack
- ReactJS
- React Router
- Tailwind CSS
- JavaScript

## Features
- Fetch products from Fake Store API
- Add / Remove products from cart
- Increase / Decrease quantity
- Dynamic total price calculation
- 10% discount on final price
- Responsive design
- Clean routing between Products and Cart pages

## Installation
npm install  
npm run dev

## API Used
https://fakestoreapi.com/products
